# Library Organizer

A Python command-line app that lets you manage a personal book library. Add books, browse your collection, and remove titles — all from a simple text menu.

## Features
- Add a book with title, author, and publication year
- List all books in your library
- Remove a book by title (case-insensitive search)
- Simple loop-based menu with a quit option

## How to run

```python
python library_sorter.py
```

No installs needed — uses only built-in Python.

## Technologies used
Python 3 (no external libraries)

## Sample books included
The app starts with three pre-loaded books:
- *The Count of Monte Cristo* — Alexandre Dumas (1844)
- *Too Late* — Colleen Hoover (2020)
- *The Housemaid* — Freida McFadden (2021)
